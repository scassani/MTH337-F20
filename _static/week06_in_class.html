from math import sqrt

def isdisjoint(x,y,R,xa,ya,Ra):
    # x,y,R are numbers
    # xa,ya,Ra are lists
    n=len(xa)
    for k in range(n):
        length = sqrt(pow((x - xa[k]), 2) + pow((y - ya[k]), 2)) # center to center distance
        radiusLength =  R + Ra[k] # sum of radii
        if radiusLength >= length:
            return False
    return True

Summary¶

numpy array

  • array must contain only one type of object
  • their size cannot be changed after creation (unlike lists where we can append, insert, pop)
  • can be indexed as lists
  • slicing won't make a copy (need to use copy())
  • support vectorial operations (unlike lists)
  • are mutable

Ex the derivative of $f$ is defined as: $$f'(x)=\lim_{h\to 0} \frac{f(x+h)-f(x)}{h}$$

so, if $h\simeq 0$ (small enough), we can say that $f'(x)\simeq \frac{f(x+h)-f(x)}{h} \quad (*)$

Approximate the derivative of $f(x)=\sin(x)$ on $[0,2\pi]$ using the formula above, and then plot it on the same graph as the real derivative

  • compute fpapp the approximated derivative using (*) (for 10 steps, 100 steps)
  • compute the real derivative fp
  • plot all 3 curves

Post the plot on the discord channel deriv.approx. with your name in the title

In [67]:
x=np.linspace(0,2*np.pi,101) # fine approx - 101 points - 100 steps
h1=x[1]-x[0] # step for fine approximation
w=np.linspace(0,2*np.pi,11) # coarse approx
h2=w[1]-w[0] # step for coarse approximation

fp=np.cos(x) # real derivative
fpapp1=(np.sin(x+h1)-np.sin(x))/h1
fpapp2=(np.sin(w+h2)-np.sin(w))/h2

plt.plot(x,fp,label='real derivative')
plt.plot(x,fpapp1,'*r',label='approx. 100 steps')
plt.plot(w,fpapp2,'k',label='approx. 10 steps')
plt.legend()
plt.show()
